# C# POS System

A simple Point of Sale (POS) system developed using C#. This project is designed to manage basic sales operations such as product management, billing, customer handling, and transaction records.

## 📌 Project Overview

The C# POS System helps small businesses manage their daily sales activities in an organized and efficient way. It allows users to add products, process customer purchases, calculate totals, and maintain sales records.

## 🚀 Features

* Product management
* Customer management
* Sales and billing
* Invoice generation
* Stock management
* Search products
* Calculate subtotal, discount, and total amount
* Save transaction records
* User-friendly interface

## 🛠️ Technologies Used

* C#
* .NET Framework / .NET
* Windows Forms / WPF
* SQL Server / Local Database
* Visual Studio

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/CSharp-POS-System.git
```

2. Open the project in Visual Studio.

3. Restore required packages.

4. Configure the database connection.

5. Run the application.

## 🧾 Main Modules

### Product Module

Manage product details such as product name, price, category, and available stock.

### Sales Module

Create bills, add products to cart, calculate totals, and complete transactions.

### Customer Module

Store and manage customer information.

### Inventory Module

Track stock levels and update quantities after each sale.
